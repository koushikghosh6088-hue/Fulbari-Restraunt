## Features Implemented
1.  **Premium Dark & Gold Design**: A modern, luxury aesthetic matching the reference image.
2.  **Responsive Layout**: Fully functional on mobile, tablet, and desktop.
3.  **Home Page**:
    *   **Hero Section**: Large parallax background with animated text.
    *   **Top Flavours**: Highlighted "Today's Special" with hover effects.
    *   **About Section**: Layered images and descriptive text.
    *   **Specialties**: Grid of cuisine categories.
    *   **Testimonials**: Customer reviews in a clean card layout.
    *   **Reservation CTA**: High-impact strip inciting action.
4.  **Menu Page**:
    *   **Dynamic Filtering**: Filter by category (Bengali, Indian, Chinese, etc.).
    *   **Real-time Search**: Search dishes by name or description.
    *   **Card UI**: Beautiful price tags, veg/non-veg indicators, and "Add to Plate" buttons (visual only).
5.  **Gallery Page**: Masonry-style grid layout showing interior and food shots.
6.  **Reservation Page**:
    *   **Booking Form**: Name, Phone, Date, Time, Guests.
    *   **WhatsApp Integration**: Submitting the form opens WhatsApp with pre-filled details.
7.  **Contact Page**:
    *   **Info Cards**: Phone, Email, Location, Hours.
    *   **Google Maps**: Interactive map embed.

## Technical Stack
-   **Framework**: Next.js 15 (React 19)
-   **Styling**: Tailwind CSS v4
-   **Animations**: Framer Motion
-   **Icons**: Lucide React
-   **Fonts**: Playfair Display (Headings), Poppins (Body)

## How to Run
The server is currently running. You can access it at:
**http://localhost:3000**
