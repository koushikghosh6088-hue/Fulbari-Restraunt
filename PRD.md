# Fulbari Restaurant Website - Product Requirements Document

## 1. Product Overview
**Product Name:** Fulbari Restaurant Website
**Goal:** Create a premium modern website for Fulbari Restaurant to showcase menu, ambiance, reservation and attract customers.

**Design Aesthetic:**
* Dark luxury theme
* Large food photography hero sections
* Card-based menu layout
* Elegant typography
* Smooth scrolling animations

## 2. Target Users
* Families looking for dining
* Couples
* Local food lovers
* People searching restaurant on Google
* Customers wanting menu before visiting

## 3. Core Pages

### ⭐ Home Page
* **Hero Banner:** Large food image, Tagline "Flavors Inspired by Tradition", Restaurant name, CTA "View Menu / Book Table".
* **Top Flavours:** Breakfast, Appetizers, Drinks, Popular Bengali dishes.
* **About Restaurant:** Short description, Ambiance photos, Balcony view highlight.
* **Best Specialties:** Bengali Thali, Chinese combos, Indian main course.
* **Testimonials:** Customer reviews.
* **Reservation CTA:** Strip.
* **Footer:** Location, Hours, Contact, Maps.

### ⭐ Menu Page
**Categories:** Bengali, Indian, Chinese, Starters, Drinks, Desserts.
**Features:** Image cards, Price tags, Veg / Non-veg badge, Search menu.

### ⭐ Gallery Page
* Food photography grid
* Restaurant interior
* Balcony seating
* Family dining photos

### ⭐ Reservation Page
**Fields:** Name, Phone, Date, Time, Guests.
**Action:** Submit → WhatsApp / Email Integration. (No login required).

### ⭐ Contact Page
* Google Maps embed
* Phone
* Address
* Parking info

## 4. UI/UX Requirements
**Design Rules:**
* Background: Dark (#0f0f0f)
* Accent: Gold
* Typography: Big, Elegant
* Cards: Rounded
* Interaction: Hover animations, Smooth scroll, Parallax hero
* Images: Full-width

**Fonts:**
* Headings: Playfair Display
* Body: Poppins

## 5. Functional Requirements
**Must Have:**
* Mobile responsive
* Fast loading
* SEO friendly
* Menu filtering
* Reservation form
* WhatsApp integration

## 6. Technical Architecture
**Frontend:** Next.js (React), Tailwind CSS, Framer Motion.
**Backend:** Lightweight (API Routes for email/whatsapp if needed).
**Hosting:** Vercel ready.

## 7. SEO Strategy
**Keywords:** Restaurant in Serampore, Bengali restaurant Hooghly, Fulbari restaurant menu, Family restaurant Serampore.
